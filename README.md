Description
Windrose Save Editor is a standalone desktop tool that lets you freely edit your Windrose character inventory. Open any save file, view all your items with icons and stats, add items from the full game database, change levels and quality, and remove unwanted items — all without touching raw game files manually.
The editor supports full EN / RU interface switching and displays item names, descriptions, and tooltips in the language you choose.

Installation instructions
Download and extract the archive to any folder (e.g. 
C:\WindroseEditor\).
Run WindroseEditor.exe.
Click Load Save and open your Windrose player save folder, in most cases C:\Users\YOUR_USER\AppData\Local\R5\Saved\SaveProfiles\76561197993424152\RocksDB\0.10.0\Players\YOUR_GUID.
Edit your inventory, then click Save to write the changes back.
Note: Always keep a backup of your save file before making changes.

Main features
📂 Load and save Windrose Player save file
🎒 Full inventory editing: add, remove, and modify items
📖 Item database with icons, stats, rarity, and category filters
🔢 Level and quality editing per item
💬 Rich tooltips: main stats, secondary stats, set bonuses, effect descriptions with real values from curve tables
🌐 English / Russian interface toggle (EN / RU button)
🔍 Search and filter items by name, rarity, and category

Requirements
Windows 10 / 11 (x64)
.NET 8 Desktop Runtime — Download from Microsoft
A copy of Windrose with an existing save file
