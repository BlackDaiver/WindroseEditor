# 🌹 Windrose Save Editor

**Windrose Save Editor** --- standalone desktop tool that lets you
freely edit your *Windrose* character inventory.

Open any save file, view all your items with icons and stats, add items
from the full game database, change levels and quality, and remove
unwanted items --- all without touching raw game files manually.

The editor supports full **EN / RU interface switching** and displays
item names, descriptions, and tooltips in the language you choose.

------------------------------------------------------------------------

## ✨ Features

-   📂 Load and save Windrose **player save files**
-   🎒 Full inventory editing: add, remove, and modify items
-   📖 Built-in item database with icons, stats, rarity & category
    filters
-   🔢 Change **level and quality** for every item
-   💬 Rich tooltips:
    -   main stats\
    -   secondary stats\
    -   set bonuses\
    -   effect descriptions with real values from curve tables
-   🌐 English / Russian UI toggle (**EN / RU** button)
-   🔍 Search and filter items by name, rarity and category

------------------------------------------------------------------------

## 📦 Installation

1.  Download the latest archive from the **Releases** page.
2.  Extract it to any folder, for example:

```{=html}
C:\WindroseEditor\
```

3.  Run:

```{=html}
WindroseEditor.exe
```

------------------------------------------------------------------------

## ▶️ How to use

1.  Click **Load Save**
2.  Open your Windrose player save folder.

In most cases it is located here:

    C:\Users\YOUR_USER\AppData\Local\R5\Saved\SaveProfiles\76561197993424152\RocksDB\0.10.0\Players\YOUR_GUID

3.  Edit your inventory.
4.  Click **Save** to write the changes back to the game.

> ⚠️ **Important:** Always keep a backup of your save files before
> editing.
